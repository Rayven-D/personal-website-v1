Bang! The Dice Game

Language: Java

Packages Used: JavaFX

IDE/Editor: NetBeans

Using Object-Oriented Programming concepts from class, we had to recreate the entire Bang! The Dice Game in Java using the original rules, characters, and die.

We had to use Java to create the game. This was also the first time many team members used GitHub, and was beneficial due to COVID-19 closing school right after spring break. We decided on splitting up the project into parts and each person worked on the separate parts.
