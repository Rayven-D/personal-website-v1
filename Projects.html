Cell Simulation

The cell simulation is a program that simulates a colony of cells growth and deaths. It compares each cell with their neighbors to determine if the cell will die or be alive in the next step, doing this to every cell for 100 steps.

This program makes use of multiprocessing to handle the arrays, and it was tested with 10,000 x 10,000 array size, only taking less than 20 minutes with 36 processes.

Health Care System

This project modeled a health care system for making appointments, changing patient data, and looking at daily reports. We had to design the software using the specified requirements.

This program makes use of Object-Oriented Programming and using JSON files to store data. We used Java to create the system, and JSON files as the "database" to store the data needed needed.

Bang! The Dice Game

This project recreated the entire "Bang! The Dice Game" on the computer. We used all the original characters, their abilites, the original dice, and the actual rules of the game

This program makes use of Object-Oriented Programming and it's conecpts. We used Java to create the system.

The program is not meant for distribution or for profit.
It was created for my CS2365 Object-Oriented Programming final project.
