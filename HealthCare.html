Health Care System

Language: Java

Classes/Packages Used: JavaFX, Gson

IDE/Editor: NetBeans

Using the UML Diagram below, the project was to build a Health Care System based on the diagram.

We had to use a Object-Oriented langauge and use a way to store user data. The team used Java since the team was more accustomed to it, and we decided upson JSON files to store user data. We used JavaFX for the GUI and Gson to read/write to the JSON files.
